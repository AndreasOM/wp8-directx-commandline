# Windows Phone 8 DirectX Command-Line Build Example

## Historical Archive (2012)

**This project is preserved for historical reference only and is no longer actively maintained.**

This repository contains a Windows Phone 8 DirectX 11 application from 2012, demonstrating command-line compilation for WP8 native applications.

## What This Is

A simple Direct3D 11 rendering application for Windows Phone 8 that can be built via:

- **MSBuild** (`msbuild.bat`)
- **Manual command-line** (`build.bat`) - Demonstrates shader compilation, linking, and XAP packaging
- **Lace build system** (`lace.bat`) - Note: The [Lace build system](https://github.com/jkoenen/lace-build) is no longer actively maintained

## Requirements

- Windows Phone SDK 8.0 (no longer available)
- Visual Studio 2012 toolchain
- Windows Kits 8.0

## Status

**Not maintained.** Windows Phone 8 was discontinued in 2014. This code is preserved for historical purposes only.
